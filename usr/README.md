# Firearms Injury Tracker

A comprehensive Flutter application designed to track and analyze incidents involving firearms. It provides a secure, user-friendly interface for logging new incidents, reviewing historical data, and visualizing trends through a dashboard.

## Features
- **Dashboard Overview:** View high-level statistics and recent incident summaries.
- **Incident Logging:** Detailed form to capture data on new incidents including date, location, injury type, and outcome.
- **Incident History:** A scrollable list of all recorded incidents with search and filter capabilities.
- **Cross-Platform:** Works on mobile, tablet, and desktop/web interfaces with a responsive design.

## Tech Stack
- Flutter
- Dart
- Material Design 3

## Setup & Run
1. Ensure Flutter is installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device.
